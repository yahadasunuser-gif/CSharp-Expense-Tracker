
💰 C# Expense Tracker
A lightweight console application built to practice C# fundamentals, focusing on logic, loops, and data validation.

✨ Features
Validation: Uses TryParse to prevent crashes from invalid inputs.

Smart Alerts: Notifies you when you've reached 50% of your budget or gone over your limit.

Multi-Input: Allows continuous expense entry until you choose to exit.

Precision: Uses decimal types for accurate financial calculations.

🛠️ How it Works
Enter your Total Budget.

Enter an Expense Name (e.g., "Coffee").

Enter the Expense Value (e.g., "5.50").

Type 'e' at any time to finish and see your final balance.

🚀 Learning Goals
Mastered nested while loops.

Implemented control flow using break and continue.

Practiced type safety and string-to-decimal conversion.
